# Projeto de Testes QA – Sistema de Login

## Objetivo
Realizar testes manuais em uma aplicação web simulando um ambiente real de QA.

## Escopo dos Testes
- Funcionalidade de login
- Validação de campos
- Tratamento de erros

## Tipos de Testes
- Testes funcionais
- Testes exploratórios
- Testes de validação

## Casos de Teste
Os casos de teste foram documentados no arquivo:
➡️ test-cases.xlsx

## Bugs Encontrados
Os bugs identificados estão documentados em:
➡️ bugs-report.xlsx

## Ferramentas Utilizadas
- Excel
- Navegador (Chrome)
- Testes manuais

## Resultado
O projeto permitiu identificar falhas no fluxo de login e melhorar a qualidade da aplicação simulada.
